  ADNOTACJA - PROGRESJE CWICZEN
Proponowane sposoby na progresje cwiczen:

pompki z podwyz. rekami -> dipy z podpartymi nogami -> dipy z asysta -> negatywy -> dipy ->
-> dipy na prostym drazku -> dipy na krazkach -> dipy plajometryczne -> dipy lucznicze ->
-> dipy lucz. na krazkach

pompki z podwyz. rekami -> negatywy -> pompki z asysta -> pompki -> diamentowe pompki -> pompki z poglebieniem ->
-> pompki japonskie -> pompki z gumami -> pompki z podwyz. nogami -> pompki plajometryczne -> pompki jednoracz ->
-> pompki glowa w dol -> pompki glowa w dol z poglebieniem -> pompki na rekach

podciaganie australijskie -> podciaganie aust. na krazkach -> podciaganie aust. z podwyzszonymi nogami -> negatywy ->
-> podciaganie z guma -> podciaganie podchwytem -> podciaganie neutralne -> podciaganie nachwytem -> podciaganie szerokie ->
-> podciaganie na krazkach -> podciaganie wysokie -> podciaganie lucznicze -> podciaganie jedna reka

sprint krotki -> sprint -> sprint pod gorke -> sprint ze spadochronem oporowym

przeskoki z nogi na noge -> wykroki plajometryczne -> przysiady bulgarskie plajometryczne -> steppery plajometryczne ->
-> steppery plajometryczne boczne -> skoki na skrzynie jednonozne -> skoki na skrzynie jendonoz. boczne

skoki obunoz w dal -> przysiady z wyskokiem -> wybicia z kolan na nogi -> skoki na skrzynie -> skok w dal+na skrzynie ->
-> przysiady z wyskokiem+wyskok na skrzynie
